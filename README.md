# 🧮 Smart Python GUI Calculator

A fully functional, modern Graphical User Interface (GUI) calculator built using Python and the Tkinter library.

## ✨ Features

- **Basic Arithmetic:** Addition, Subtraction, Multiplication, and Division.
- **Advanced Operations:** Square root (√) and Percentage (%) calculations.
- **Precision Control:** Real-time calculation using Python's `eval()` function.
- **User-Friendly UI:**
  - Responsive grid layout that stretches with the window.
  - "Back" button (⌫) to delete the last character.
  - "Clear" (CE) button to reset the display.
  - Right-aligned text input for a professional feel.
  - Custom color coding for operators and results.

## 🚀 How to Run

To run this calculator on your local machine, follow these steps:

1. **Prerequisites:** Make sure you have Python installed. (Tkinter usually comes pre-installed with Python).
